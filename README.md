# Calculator App

This is a simple calculator project built with **HTML, CSS and JavaScript**.

I created this project around five years ago while I was practicing the fundamentals of web development. The goal was to understand how HTML, CSS and JavaScript work together to build an interactive web application.

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Overview

The calculator allows basic arithmetic operations and was one of my early practice projects while learning frontend development.

Even though it is a simple project, it represents an important part of my learning process because it helped me practice:

- HTML structure
- CSS styling
- JavaScript logic
- DOM manipulation
- User interaction in the browser
